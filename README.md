# devonian
Devonian is a minecraft mod for hypixel skyblock aimed towards enhancing your gaming for minecraft 1.21.5

# Features
## Dungeons
* BoxStarMobs
* AutoRequeueDungeons
* ExtraStats
* MimicKilled
* CryptsDisplay
* DeathsDisplay
* MilestoneDisplay
* PuzzlesDisplay
* RemoveDamageTag
* HideNoStarTag
## Garden
* GardenDisplay
* PestsDisplay
